# ROOKI Support Website

A simple and modern support website for ROOKI, a gamified social media platform designed for sports fans.

## Features

- **Homepage**: Welcome section with information about ROOKI
- **Support Center**: Quick access cards for different support topics
- **FAQ Section**: Expandable accordion with common questions
- **Contact Section**: Multiple ways to get in touch with support
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Smooth Animations**: Interactive elements with smooth transitions

## Files

- `index.html` - Main HTML file
- `styles.css` - All styling and responsive design
- `script.js` - Interactive functionality (FAQ accordion, smooth scrolling)
- `README.md` - This file

## Usage

Simply open `index.html` in a web browser to view the website. No build process or dependencies required.

## Customization

You can easily customize:
- Colors in the `:root` CSS variables
- Content in the HTML file
- FAQ questions and answers
- Contact information
